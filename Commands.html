GitHub Commands

1

Download Git

We need to download and install Git in our systems to use it. Visit the official website https://www.git-scm.com/downloads to download Git. Choose the type of your system and then download Git.

After downloading install Git.

If you already have Git installed then you can update to the latest version.

2

Check if Git is downloaded

Run the following command in your code editor terminal and check if it shows you the version of Git you are using.

git --version

3

Initialize a Git Repository

After you complete writing your code and want to save it in Git. First go to your terminal, then navigate to the folder in which your project is located and then run the following command to initialize an empty Git repository.

git init

4

Add files to Git Repository

Next you have to run the following code to add the desired files to your empty Git repository.

git add file name

5

Check the status of your Git Repository

Once you have added the desired files to your Git repository, you have to check the status of your files. Run the following command to check if all the files you staged to the Git repository got staged in your repository.

git status

The files that got staged will be displayed in green and the files displayed in red are unstaged files.

6

Commit the staged files

Next you have to commit the staged files to your repository. Run the following command to commit the changes to the repository.

git commit -m 'your commit message.'

7

Create repository on GitHub

Visit https://github.com/ and create an account if you don't have an account or sign in to your account. Create a new repository.

8

Create repository on GitHub

In the previous step you have created a remote repository. now you have to add the local repository t=you created to the remote repository. This will allow you to push and pull changed fron this repository itself. Following command will do this.

git remote add origin your remote repository link

Note:

If you are doing this for the first time, then you will be getting an error "Please tell me who you are."

Run the following commands to solve this error.

git config --global user.email"your email" git config --global user.name"your name"

9

Rename the current branch

Run the following command to rename your current branch. "main" is the default branch. By default the file commited saves to the main branch.

git branch -M main

10

Push local branch to remote repository

Now run the following command which will push your local repository to the remote repository. git push will push your code. origin is the name of the remote repository to push your code to.

git push -u origin main

Check the commit history

Run the following command if you wish to check the commit history./p>

git log
